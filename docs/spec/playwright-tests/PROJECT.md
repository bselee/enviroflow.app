## Objective
Create comprehensive Playwright E2E tests for all app pages with focus on data display validation and controller connectivity to controlled devices.

## Success Metrics
- [ ] All pages have dedicated test suites
- [ ] Dashboard data display tests pass
- [ ] Controller connectivity tests validate device control
- [ ] Sensor readings display correctly
- [ ] Real-time updates work properly
- [ ] No critical bugs in data flow

## Scope
**In scope:**
- Dashboard page tests (environment snapshot, timeline, alerts)
- Controllers page tests (add, list, delete, connect)
- Controller detail page tests (device control, sensor display)
- Settings page tests (profile, notifications, preferences)
- Automations page tests (list, create, toggle)
- Schedules page tests (create, edit, activate)
- Authentication flow tests
- Data display validation
- Controller device connectivity

**Out of scope:**
- Performance/load testing
- Cross-browser testing (Chromium + Mobile only for MVP)
- API unit tests (separate effort)
